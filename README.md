# Welcome to my basic-ecommerce app with vue,vuex,vue-router.

## To install 
```
cd vue-basic-ecommerce
npm install
```

### To watch your changes instantly 
```
NODE_ENV=development

npm run serve
```
### Run for development
```
NODE_ENV=development

npm run serve
```
###

### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
